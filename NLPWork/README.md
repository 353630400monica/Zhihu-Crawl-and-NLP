
# 介绍
## 关注数、浏览数、回答数之间的线性关系
>工具：matplotlib、sklearn
>利用爬取到的数据，使用matplotlib来绘制三者的平面分布图，并使用sklearn来对三者之间的关系进行线性关系的预测<br>

## 关键词的词云
>工具：jieba分词、wordcloud 、numpy<br>
>选取爬取到的问题中回答数较高的问题，提取其中的关键词，形成词云<br>
